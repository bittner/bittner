# Star History Chart

This directory contains the automatically generated Star History Chart for popular projects maintained or co-maintained by @bittner.

## Files

- `star-history.svg` - The star history chart showing cumulative stars over time for multiple repositories

## How it's Generated

The chart is automatically generated by the [Star History Chart workflow](../.github/workflows/star-history.yml) which:

1. Runs weekly on Sundays at midnight UTC
2. Can be manually triggered via workflow_dispatch
3. Fetches star data from GitHub API for all configured repositories
4. Generates an SVG chart using matplotlib
5. Commits the updated chart back to this directory

## Manual Update

To manually trigger a chart update:

1. Go to the [Actions tab](../../actions/workflows/star-history.yml)
2. Click "Run workflow"
3. Select the branch and click "Run workflow"

The chart will be updated within a few minutes.
